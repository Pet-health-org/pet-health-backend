# PetHealth - Sistema de Gestión Veterinaria

Sistema de gestión veterinaria para clínicas de animales. Permite administrar propietarios, mascotas, citas, historias clínicas, vacunación, inventario, notificaciones y reportes.

## Módulos del Sistema

1. **Gestión de Propietarios** (HU-01 a HU-03)
2. **Gestión de Mascotas** (HU-04 a HU-06)
3. **Gestión de Citas** (HU-07 a HU-09)
4. **Historia Clínica y Consultas** (HU-10 a HU-12)
5. **Gestión de Vacunación** (HU-13 a HU-15)
6. **Gestión de Inventario** (HU-16 a HU-18)
7. **Notificaciones** (HU-19 a HU-21)
8. **Reportes y Estadísticas** (HU-22 a HU-24)
9. **Seguridad y Control de Acceso** (HU-25 a HU-27)
